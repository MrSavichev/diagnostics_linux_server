# diagnostics_linux_server

Скрипт диагностики сервера Linux Debian. Собирает данные о состоянии системы и сохраняет отчёт в текстовый файл.

## Что проверяет

| Категория | Что именно |
|-----------|-----------|
| **Система** | ОС, версия ядра, аптайм, таймзона |
| **Железо** | CPU (модель, ядра, частота), RAM, swap |
| **Диски** | Использование разделов (df) |
| **Нагрузка** | Load average, топ процессов по CPU |
| **Сеть** | Сетевые интерфейсы, слушающие порты |
| **Безопасность** | Файрвол (UFW / iptables), последние входы, auth.log |
| **Сервисы** | Запущенные и упавшие systemd-сервисы |
| **Логи** | auth.log, syslog (последние 20 строк) |
| **Обновления** | Список доступных пакетов apt |
| **Docker** | Список контейнеров (если установлен) |

## Требования

- Python 3.6+
- Linux Debian / Ubuntu
- Для полного доступа к логам рекомендуется запуск через `sudo`

## Установка

```bash
git clone https://github.com/MrSavichev/diagnostics_linux_server.git
cd diagnostics_linux_server
```

## Использование

### Обычный запуск
```bash
python3 diagnostics.py
```

### С правами sudo (рекомендуется — откроет доступ к логам)
```bash
sudo python3 diagnostics.py
```

### Только сохранить отчёт без вывода в консоль
```bash
python3 diagnostics.py > /dev/null
```

## Вывод

Скрипт:
1. Выводит прогресс проверок в консоль
2. Печатает полный отчёт в консоль
3. Сохраняет отчёт в файл `report_YYYYMMDD_HHMMSS.txt` в текущей директории

Пример имени файла: `report_20260319_143022.txt`

## Пример отчёта

```
==============================================================
  DEBIAN SERVER DIAGNOSTICS REPORT
  Host    : my-server
  Date    : 2026-03-19 14:30:22
==============================================================

──────────────────────────────────────────────────────────────
  OS / Distribution
──────────────────────────────────────────────────────────────
Distributor ID: Debian
Description:    Debian GNU/Linux 12 (bookworm)
Release:        12
...
```

## Передача отчёта

```bash
# Скопировать на локальный ПК через scp
scp user@server:~/report_*.txt ./

# Или сразу запустить и скачать
ssh user@server 'cd /tmp && python3 - < diagnostics.py'
```
